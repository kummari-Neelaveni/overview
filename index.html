HTML

About HTML

HTML stands for Hypertext Markup Language,it was invented by Tim Bernes Lee.It is used to create and structure the content on the web page.The latest version of html is HTML5.

Structure of Html

Doctype:It Defines the latest version of HTML
Root Element:It is the Main Container that holds everything on the webpage.
Head:Gives information about our document.
Body:Displays the content on the browser.

Tags

The tag which is used to define the structure and content of a webpage and it is enclosed between <,> brackets to dispaly data in browser.
Syntax: <tagname> </tagname>
<tagname> or <tagname/>
There are two types of tags namely:
1.Paired Tag:Paired tag or container tag.An element which has both opening and closing tag is known as Paired tag.
ex:h1-h6,p,div.
2.Unpaired Tag:A tag which closes itself is called Unpaired tag.
ex:hr,br,meta.

Elements

The Combination of Opening & Closing and with content between the tags is known as Elements.There are three types of Elements those are
1.Block-level Element:The tag which occupies whole width and creates a new line.
ex:h1-h6,p,div,semantic elements.
2.Inline ElementThe tag which occupies only required space and does not create a new line,in css we can apply a width,height,padding top & bottom,margin top & bottom.
ex:span,strong,b,i.
3.Inline-Block elements:The Element which occupies only required space and does not create new line and we can apply width,height,margin and padding properties.
ex:img,input,iframe,button.
